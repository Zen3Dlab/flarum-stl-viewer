# flarum-stl-viewer

A [Flarum](http://flarum.org) extension which adds an STL viewer BBCode. Simply type: `[STL]{url}[/STL]`.

## Installation

Install with composer:

```sh
composer require zen3d/stl-viewer:"*"
```

## Updating

```sh
composer update zen3d/stl-viewer:"*"
php flarum migrate
php flarum cache:clear
```
